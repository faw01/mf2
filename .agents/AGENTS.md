# mf²

SaaS monorepo built with Next.js 16, Convex, and Turborepo, managed with **bun** as the package manager.

## Commands

```bash
bun install     # Install dependencies
bun run dev     # Start all apps
bun run build   # Build all apps and packages
bun run test    # Run all tests
bun run fix     # Format and lint (ultracite); lefthook runs this on pre-commit
```

## Skills

Vendored skills in `.agents/skills/` cover the patterns and conventions for this repo (Convex, Clerk, Expo, Stripe, shadcn, Turborepo, and more). Read the matching skill before working in an area.

Conventional commits, single line only: `type(scope): description`. No body, no footer.
